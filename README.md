## Overview

This repository hosts the book:

> **Heretikal™ Physics Presents:**  
> **Heretikal Physics (Volume 1 – Introduction to Basic and Early Atomic Physics – Part 1)**  
> by **Mark Thomas Chatham**, published in 2024.

The book is intended as an introduction to basic physics and early atomic physics, with an emphasis on clear conceptual development and mathematically consistent derivations. Equations are typeset in LaTeX within the source, and the publicly distributed format here is PDF.
